Ціна 23 000 грн

Ціна може змінюватись залежно від вхідної ціни на запчастини

| ТТХ                           |                |
| ----------------------------- | -------------- |
| Протестована дистанція роботи | 5.5 км         |
| Прийом відео                  | 5,8 Ггц        |
